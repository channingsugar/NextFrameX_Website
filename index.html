# 为什么是 NextFrame X？

> 胶片冲扫工作流程里最费时的几件事：边框识别、色罩去除、色偏修复、亮度补偿。NextFrame X 通过 Metal 与 CoreImage 管线，把这些步骤自动化，同时保留能让暗房玩家满意的手动控制。

## 功能速览

  • 智能色罩/色偏去除 ：阶调检测 + 三通道独立校正，支持手动微调。
  • 边框/裁切自动识别 ：支持 35mm、120 以及自定义画幅，键盘 A 即可框选。
  • 一键亮度修复 ：基于直方图分位的亮度补偿，暗卷也能恢复层次。
  • 批处理与参数快照 ：可对整卷胶片批量推送调整，随时撤销/恢复。
  • 16-bit 导出 ：保持 Linear sRGB 工作空间，最大程度保留底片信息。

## 斋戒般的工作界面

  • ⇧ + Enter ：全屏预览，对比前后。
  • [ / ] ：在七档风格之间快速切换。
  • Command + C / V ：复制/粘贴整套调整。
  • 任何一步都可以撤销，真正的「所见即所得」。
